The following guides help you use Chronosphere Observability Platform features to
perform a given task.

## Reduce high cardinality

When your metric cardinality is high, you can
[use this guide](/control/shaping/reduce-cardinality) to learn how to reduce it.

## Explore services and telemetry data

Use [Service pages](/observe/services/service-exploration) to explore issues with a
service and connected services, and leverage other telemetry data to help find and
resolve problems.

## Use traces to find problems

Learn how to use tracing data during
[on-call triage](/investigate/querying/traces/examples) to find the source of a
problem.

## Best practices when using dashboard intervals

Learn about dashboard [query intervals and data resolution](/observe/dashboards/customization/intervals).
