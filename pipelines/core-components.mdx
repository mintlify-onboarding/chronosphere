Each release of Chronosphere Telemetry Pipeline includes several components with
their own distinct version numbers. To complete certain tasks, you might need to
reference the latest version of a particular component rather than the latest version
of Telemetry Pipeline itself.

<Info>
To compare versions of Fluent Bit and Core Agent, see
[Version equivalents](/fleets/agent/version-equivalents).
</Info>
