In Chronosphere Telemetry Pipeline, you can use [fleets](/fleets) to manage instances
of Fluent Bit and Core Agent. Core Agent is based on Fluent Bit, but Core Agent
updates on its own cadence and uses a separate versioning system, which means
the version numbers between the tools aren't identical.
