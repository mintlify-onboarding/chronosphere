Chronosphere Observability Platform lists all triggered alerts and their status,
and links to a [detailed visualization](/investigate/alerts/alert-details) of each
alert.

To view the list of triggered alerts, select
**<Icon icon="bell" /> Alerting <span aria-label="and then">></span> Alerts**
in the navigation menu.

## View an alert's status

Each listed alert includes several columns of information:

- The date and time that it was **Created at**
- Its **Status**, designated with an icon, text, and color:

  | Icon | Text | Description |
  | ---- | ---- | ----------- |
  | <Icon icon="hexagon-exclamation" style={{ color: 'rgb(238, 108, 108)' }}/>      | **Critical** | Actively triggered alert that exceeds the defined critical conditions. |
  | <Icon icon="triangle-exclamation" style={{ color: 'rgb(255, 178, 73)' }}/>               | **Warning**  | Actively triggered alert that exceeds the defined warning conditions.  |
  | <Icon icon="circle-info" style={{ color: 'rgb(166, 191, 248)' }}/> | **Muted**    | Alert that's muted by an active muting rule.                           |
  | <Icon icon="circle-check" style={{ color: 'rgb(89, 204, 141)' }}/>         | **Resolved**  | A resolved alert.                                                      |
- Its title (**Alert**), as a link to its [alert details](/investigate/alerts/alert-details) page
- The duration that the alert has been **Alerting for**
- The **Configuration entity** related to the alert, as a link to the associated
  entity, such as a [monitor](/investigate/alerts/monitors) or [SLO](/observe/slo)

You can define the number of **Rows per page** to display by clicking its dropdown
and selecting a value. To navigate between pages, click the **\< Go to previous page**
and **> Go to next page** buttons.
