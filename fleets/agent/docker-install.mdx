In Chronosphere Telemetry Pipeline, you can use [fleets](/fleets) to manage instances
of Core Agent.

Use the following command to install Core Agent in Docker:

```shell /VERSION/
docker run --rm -it ghcr.io/chronosphereio/calyptia-core-agent:VERSION
```

Replace _`VERSION`_ with the [latest version](/fleets/agent/version-equivalents)
of Core Agent.
